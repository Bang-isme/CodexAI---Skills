## Usage Notes
- Use one palette family per page or screen cluster; do not mix rows.
- For dark themes, keep page background and surface within the same family so shadows read as depth, not dirt.
- For light themes, rely on surface tone shifts and accent restraint instead of stronger shadows.
- If a brief explicitly asks for "premium", start with `Obsidian Tide`, `Linen Theory`, `Pearl Slate`, or `Oatmilk Studio`.
